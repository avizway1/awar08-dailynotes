cd : Change directory
ls : displays the content we have in the directory (files/folders)
pwd : Print working working (tells you where you are now)

cd /



In linux everything starts from "/"

/ : Root Directory : Top level directory, the parent of all ither directories in Linux OS.

/root : The home directory for root user. (superuser)
windows: similar to C:\Users\Administrator

/home : Home directory of regular/non-root users. 
Windows: c:\users\<username>

/bin : Binary Executables : Contains all essential binary files used by all users (ex: ls, cp, mv).. 

/boot : Boot loader files : Contains the linux kernel, boot configs and its related files.

/dev : Device Files : Contains files that represent hardaware devices like hard disks. 
windows: device manager

/etc : Configuration files : Stores system wide configuration files (i.e; /etc/passwd, /etc/resolv.conf)
*Service management and system settings

/usr : user Programs : Contains uuser level programs and libs. 

/sbin : System binaries : contains system admin commands i.e; ifconfig, iptables
*Only root or superusers can execute these commands.

/var : Variable files : Contains data that changes frequently. (logs, databases, documentroot Path)

/lib & lib64 : Essential shared Library files.. 
* similaar to windows .dll files

/proc (Process information) : COntains system runtime information (CPU / RAM / processes)

/opt : Optional Software 
/mnt : Mount points : 
/media : removable media :

vim : VI Improved editor : 

Command Mode : Used for navigating, deletion, other file operations.

Insert Mode : Used for entering text into the file.

TO switch:

command mode ---> Insert Mode : Press "i"
Insert Mode ---> COmmand Mode : Press "esc"

To Saving/exiting:

:wq		: Save changes and quit editor
:q!		: Quit editor without saving changes.

o 		: Open a new line below the current line and enters the insert mode

k		: Move cursor up one line
j 		: move cursor down one line

x		: Delete the charactor

dw		: Delete from the cursor to the end of current word. (left side word)
db		: Delete the word before the cursor. (right side word)
dd		: Delete the entire current line.

100dd	: delete the next 100 lines from the cursor
